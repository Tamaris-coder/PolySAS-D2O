# PolySAS-D2O

Automated SasView pipeline for polymer nanoparticles in heavy water (D₂O)  
Open-source SANS modeling toolkit for drug delivery systems · 2025

### Features
- Instant core-shell micelle & polymersome models  
- Real SANS datasets (ILL-D22, NIST)  
- Publication-ready plots in <10 seconds  
- 3D animations + batch fitting
